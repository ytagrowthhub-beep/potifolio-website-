# Project Images Directory

This folder should contain images for your projects.

## Required Images

For each of the 6 projects, you'll need:

### Project 1: E-Commerce Platform
- `ecommerce-thumb.jpg` - Thumbnail image (800x600px recommended)

### Project 2: Task Management App
- `taskapp-thumb.jpg` - Thumbnail image (800x600px recommended)

### Project 3: Social Media Dashboard
- `social-dashboard-thumb.jpg` - Thumbnail image (800x600px recommended)

### Project 4: Fitness Tracking App
- `fitness-thumb.jpg` - Thumbnail image (800x600px recommended)

### Project 5: Real Estate Platform
- `realestate-thumb.jpg` - Thumbnail image (800x600px recommended)

### Project 6: Learning Management System
- `lms-thumb.jpg` - Thumbnail image (800x600px recommended)

## Image Guidelines

- **Format**: JPG or PNG
- **Size**: 800x600px or larger (16:9 or 4:3 aspect ratio)
- **Optimization**: Compress images before uploading for better performance
- **Content**: Screenshots, mockups, or relevant project visuals

## Adding Images

1. Create the `/public/projects/` folder if it doesn't exist
2. Add your project images with the names above
3. The images will automatically appear in the project cards and detail pages

## Fallback

If images are not found, the system will show a placeholder with a code icon.
